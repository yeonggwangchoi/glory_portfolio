About

About Me

안녕하세요. AI를 활용해 자율주행의 한계를 극복하고자 하는 지원자 최영광입니다.

현재 성균관대학교 자동화연구실에서 석사 과정 중이며, 자율주행·자율주차·Computer Vision 기반 자동화 시스템을 중심으로 연구하고 있습니다.

자율주행은 다양한 기술이 융합되는 분야인 만큼, 연구 성과를 실제 시스템으로 구현할 수 있는 엔지니어의 역할이 중요하다고 생각합니다. 제게 도전은 항상 두려움보다는 설레는 마음이 컸기 때문에 도전을 통해 경험하는 것을 좋아하고, 그 경험을 기반으로 성장하는 것을 즐깁니다. 이러한 마음가짐으로 석사 과정 동안 국제 학회 논문 5편을 발표하며 학술적 전문성을 체계적으로 쌓아 왔고, 학부 시절에는 자율주행 관련 경진대회 수상 경험을 통해 실전 중심의 개발 역량을 축적해 왔습니다.

특히 저는 빠르게 변화하는 기술 흐름을 정확히 포착하고, 새로운 기술을 신속히 학습해 실제 문제 해결에 적용하는 데 강점을 가지고 있습니다. 이를 바탕으로 연구와 개발 간의 간극을 줄이고, 현장에서 바로 활용 가능한 솔루션을 구현해 왔습니다.

앞으로도 이러한 연구 전문성과 개발 역량을 바탕으로, 미래 모빌리티 기술을 현실로 구현하는 엔지니어로 성장해 나가고자 합니다.

인프라 센서 및 V2X 통신 기반의 효율적인 자율 발렛 주차(AVP) 시스템 개발

연구원
기술 스택 :
- OS: Linux (Ubuntu 22.04)
- Languages: C++, Python
- Frameworks: ROS 2 (Humble)
- Libraries: PyTorch (YOLOv11), OpenCV
- Simulation: Gazebo, AWSIM
- Hardware: FishEye Cameras, V2X Module
주요 기능 :
1. 시뮬레이션 환경
- YOLOv11 기반 다중 키포인트(Keypoint) 검출: 차량의 전/후방 및 사이드미러 등 4개 주요 지점을 검출하여 3D Bounding Box 대비 연산 효율이 높은 2D 기반 자세 추정 모델 구축
- 계층적 헤딩(Heading) 추정 알고리즘: 기하학적 중복성(Geometric Redundancy)을 활용한 6단계 우선순위 로직을 설계하여, 폐색(Occlusion) 상황에서도 안정적인 방향 추정
- 분산형 서버-클라이언트 아키텍처: 고부하 인식(Perception)은 서버에서, 실시간 제어(Planning/Control)는 차량(Client)에서 수행하는 구조 설계로 CPU 부하 최적화 및 시스템 확장성 확보
- Kinematics 기반 경로 계획 및 제어: Hybrid A-star를 통한 비홀로노믹(Non-holonomic) 경로 생성 및 MPC를 이용한 정밀 경로 추종(10Hz) 구현
- V2X 통신 기반 위치 동기화: 서버에서 추정한 전역 위치(Global Pose)를 차량으로 전송하고, 통신 단절 시 IMU/Odometry 기반 추측 항법(Dead Reckoning)으로 전환되는 하이브리드 위치 추정 로직 실현
- 실시간 리소스 분석: 시스템 구동 시 차량 측 추가 CPU 점유율을 2% 내외로 유지하며 저사양 온보드 컴퓨터에서도 구동 가능한 최적화 달성
2. 실제 환경
- 실제 주차 환경 구축 (16m x 14.5m): 시뮬레이션에서 검증된 로직을 실제 물리 환경에 포팅하여 시스템의 실효성 검증
- 광각 FishEye 카메라 활용 및 최적화: 180도 광각 카메라를 도입하여 인프라 사각지대를 최소화하고, 단일 카메라로 넓은 영역을 커버하는 효율적인 센서 인프라 구축
